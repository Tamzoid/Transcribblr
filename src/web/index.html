Bulk clear

  

    Both actions touch every record in this project. They do not
    delete the records themselves or the timing — only the text fields.
    Speakers, scenes, characters, and the rest of the project context are
    untouched.
  


  

    
Clear all translations

    

      Removes the English translation, literal lane, translator notes, and
      the 🆕 review flag from every record. Keeps Japanese + romaji intact.

      Reverts every record to its untranscribed state — Japanese back to
      ????, romaji + English + literal + notes wiped. The
      transcribe pipeline can re-fill them. Destructive: cannot be undone
      without a project ZIP backup.
